# Agent Documents

This folder stores documents generated by AI agents during workflow execution.

## Naming Convention

Files are named as: `{COMMAND}-{slug}.md`

| Command        | Example File                |
| -------------- | --------------------------- |
| `/plan`        | `PLAN-user-auth.md`         |
| `/brainstorm`  | `BRAINSTORM-caching.md`     |
| `/create`      | `CREATE-blog-comments.md`   |
| `/update`      | `UPDATE-dark-mode.md`       |
| `/debug`       | `DEBUG-api-500.md`          |
| `/design`      | `DESIGN-payment-service.md` |
| `/test`        | `TEST-auth-service.md`      |
| `/orchestrate` | `ORCHESTRATE-full-stack.md` |

## Slug Generation Rules

- Extract 2-3 key words from topic
- Lowercase, hyphen-separated
- Maximum 30 characters

## Note

This folder is excluded from git by default (via `.git/info/exclude`). Your documents are preserved during `agk update`
and `agk remove`.
